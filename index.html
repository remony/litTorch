Welcome to LitTorch

LitTorch is a lightweight basic Android app that allows you to activate the flash on your phone or tablet instantly.

The focus

If your phone/tablet does not give you the option to activate a torch you are left with apps that are on the store, the focus is performance, there is no graphics and all the functionality is performed code side so there is nothing major to affect the performance when opening the app.

Installing

Because this app is not on the play store you are left to install it yourself, you can grab the latest release here or you can clone/download this project and compile it yourself.
